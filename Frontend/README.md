## mBook
Simple book searching web app just for fun. 
### Tech Stack
- React 
-Html
-Css
-JavaScript
### Features
- Search for books by title, author, or ISBN.
- View book details including title, author, cover image, and description.
- Responsive design for mobile and desktop.
